Title: Cloud Foundry Pictures Presents an Animated Journey through the Cloud Native Galaxy
Publication date: 2018-10-12
Playlist: Cloud Foundry Summit Europe 2018
Description: 
	This inspired cartoon helps you navigate the Cloud Foundry solar system with a journey through the Cloud Native Galaxy! Learn how the Cloud Foundry Application Runtime planet is at the center of an intergalactic ecosystem of planets, comets, stars and an asteroid belt comprised of related technologies like the Open Service Broker API, Kubernetes, Cloud Foundry Container Runtime, Istio, Envoy and many, many more. 

The Cloud Native Galaxy is as limitless as space itself. Join the adventure!

All drawings are original work of Denise Yu. Animated by Rebecca Silvers (www.rebeccasilvers.com) with original music by Jesse O’Connell. Written and directed by Caitlyn O’Connell. Distributed by Cloud Foundry Foundation.
