Title: GitHub Satellite India 2021 - Journey of a project made in India: Hoppscotch
Publication date: 2021-03-28
Playlist: GitHub Satellite India 2021 - Open Source: Day 1
Description: 
	Presented by Liyas Thomas, Creator @ Hoppscotch

Hoppscotch is an open source request builder and API testing tool used by over 150,000 developers globally. In just a short period of time, it has surged in popularity and become widely adopted by the developer community. Starting off with his motivation behind creating Hoppscotch, Liyas Thomas will share his journey of building the software and his methodologies for scaling it. He'll also dive into how he manages to financially sustain an open source project. Finally, based on his experiences with the open source community and developers worldwide, Liyas will share his insights on the future of open source developer tools.
