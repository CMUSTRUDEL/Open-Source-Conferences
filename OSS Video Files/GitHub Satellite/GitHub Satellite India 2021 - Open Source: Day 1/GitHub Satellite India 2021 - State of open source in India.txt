Title: GitHub Satellite India 2021 - State of open source in India
Publication date: 2021-03-28
Playlist: GitHub Satellite India 2021 - Open Source: Day 1
Description: 
	Presented by Gaurav Godhwani, Director & Co-Founder @ CivicDataLab

Open source is increasingly changing the way software is developed and how developers collaborate. The outcome of this is seen not just in the software industry, but across various industries, verticals and domains. In this session, Gaurav talks about findings from a recent research report on how open source is helping make an impact across various domains in India and share some interesting insights on the way open source operates in India.
