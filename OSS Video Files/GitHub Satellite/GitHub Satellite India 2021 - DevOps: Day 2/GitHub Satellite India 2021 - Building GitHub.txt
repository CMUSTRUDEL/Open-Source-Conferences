Title: GitHub Satellite India 2021 - Building GitHub
Publication date: 2021-03-28
Playlist: GitHub Satellite India 2021 - DevOps: Day 2
Description: 
	Presented by Sanjay Malpani, VP - Software Engineering @ GitHub

Millions of developers and companies build, ship, and maintain their software on GitHub, making it the largest and most advanced development platform in the world. With such a massive scale comes massive engineering challenges. In this session, Sanjay Malpani, the VP of Software Engineering at GitHub, will give you an overview of how the team builds GitHub and the engineering culture at the company. Finally, he'll leave you with some tips on how you can use GitHub to build better software.
