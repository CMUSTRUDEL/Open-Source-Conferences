Title: #ACEU19: Sergey Beryozkin – Apache Tika Goes Native with GraalVM and Quarkus
Publication date: 2019-10-26
Playlist: ApacheCon Europe 2019 – Berlin
Description: 
	More: https://aceu19.apachecon.com/session/apache-tika-goes-native-graalvm-and-quarkus

This session will start with an overview of Apache Tika and show how Quarkus helped to prepare it to run in a GraalVM native image to give its users much faster startup and parse response times and explain what changes may have to be applied for a given Apache project to become GraalVM native image ready.
