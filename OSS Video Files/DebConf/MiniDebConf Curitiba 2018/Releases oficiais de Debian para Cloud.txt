Title: Releases oficiais de Debian para Cloud
Publication date: 2018-05-29
Playlist: MiniDebConf Curitiba 2018
Description: 
	by Helen Koike

At: MiniDebConf Curitiba
https://minidebconf.curitiba.br/2018/

Provedores de cloud como a AWS (Amazon), Azure (Microsoft), Google Cloud
Engine e diversos outros oferecem a opção de deploy de images de várias
distros incluindo o Debian. Atualmente cada provedor disponibiliza uma imagem
própria independente da comunidade e o time de Cloud do Debian quer mudar
isso.

Com releases de imagens oficiais para diversas plataformas, os usuários
poderão ter mais confiança na imagem que usam sabendo que passou pelo nosso
Q&A sendo devidamente testado, contendo apenas pacotes que seguem o DFSG
(Debian Free Software Guidelines). Além de criar um ecosistema para que os
provedores possam trabalhar colaborativamente e contribuir de volta para a
comunidade. Nesta palestra Helen vai apresentar os planos atuais de
infraestrutura para dar suporte a imagens oficiais de Debian para Cloud, o
estado atual e o que falta.
