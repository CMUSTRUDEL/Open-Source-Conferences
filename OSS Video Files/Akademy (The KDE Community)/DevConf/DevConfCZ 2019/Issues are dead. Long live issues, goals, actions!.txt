Publication date: 2019-03-04
Playlist: DevConfCZ 2019

	Presentation name: Issues are dead. Long live issues, goals, actions!
Speakers: Fernando Colleone, Adam Šamalík

Description:  Running an open source project but work tracking feels tricky? Is your issue tracker too messy? Come and learn about Agile and a specific approach using issues, goals, and actions to organize and track user feedback, objectives, and work to be done.

We'll also have two specific examples of open source project using this system to organize what they're doing — Fedora Docs and Fedora Modularity.

You'll learn what an Agile mindset is, and about one specific approach you can take.
