Title: How do algorithms work?

Description: 
	Algorithms explained. Learn more about about algorithms on IRL, an original podcast from Mozilla: https://mzl.la/2FwptK6

"Blame the algorithm" is no longer an excuse. From Google search to Facebook news, algorithms shape our online experience, and like humans, algorithms are flawed. Whether intentional or not, we write biases into code. It’s up to us to push for accountability. Because when bad code spreads disinformation, it’s never something that “the algorithm did.” It’s something people did.
