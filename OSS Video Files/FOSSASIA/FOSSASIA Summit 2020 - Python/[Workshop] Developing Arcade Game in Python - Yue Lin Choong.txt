Title: [Workshop] Developing Arcade Game in Python - Yue Lin Choong
Publication date: 2020-03-28
Playlist: FOSSASIA Summit 2020 - Python

	In this workshop, we will dive into the popular PyGame library. First we will learn about basic arcade game architecture:

- Game loops
- Game stats
- Game objects
- Game play

Then Yue Lin will go through how she developed a Typing Tutor Game using PyGame.
