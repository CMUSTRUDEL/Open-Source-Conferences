Title: Manage Events, Attendees and Tickets using Open Event Orga Android App - Areeb Jamal
Publication date: 2017-09-03
Playlist: FOSSASIA GSOC 2017
Description: 
	Open Event Orga Android Application allows event organizers to manage their events easily and track sales, attendee check ins, tickets and much more right on their devices

Features include:
- Sales overview with graphs
- More fluent QR scan workflow
- Background job scheduling for offline support
- Event and Organizer Info
- Uniform and modern material look

Github Repo - https://github.com/fossasia/open-event-orga-app
