Title: Open Event Organizer | FOSSASIA | Amol Mejari
Publication date: 2017-09-03
Playlist: FOSSASIA GSOC 2017
Description: 
	This is a screencast of Open Event Organizer, an android application for event organizers and entry managers created by FOSSASIA. It is a client side application for another project under FOSSASIA named Open Event Organizer Server.

The app is developed as a part of Google Summer of Code 2017 under the mentorship of FOSSASIA.
