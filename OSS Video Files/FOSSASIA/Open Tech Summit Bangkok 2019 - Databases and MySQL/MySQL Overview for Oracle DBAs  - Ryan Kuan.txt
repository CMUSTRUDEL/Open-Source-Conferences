Title: MySQL Overview for Oracle DBAs  - Ryan Kuan
Publication date: 2019-10-19
Playlist: Open Tech Summit Bangkok 2019 - Databases and MySQL
Description: 
	Ryan Kuan - MySQL

MySQL is the most popular open source database used and trusted by millions of users for cloud, internet and mobile applications. More and more enterprises are evaluating open source database technology as alternatives to their existing Oracle database. In this session, we will help existing Oracle DBAs to understand MySQL better as a viable database platform to  complement existing Oracle database.
