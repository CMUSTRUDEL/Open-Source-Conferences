Title: Garrett Honeycutt - Security compliance testing with Inspec
Publication date: 2019-06-18
Playlist: 2019 SouthEast LinuxFest
Description: 
	SouthEast Linux Fest 2019
Learn how to take security policies from documents that no one reads and convert that into code using Inspec. This talk will explain Compliance as Code and how to implement this practice. Are your systems secure? What about cloud resources? We will use Inspec to functionally test our environment as well as discuss how to implement this as part of a CI/CD pipeline.
