Title: Dr. Hafiz Hanif: WordPress Theme for Academics
Publication date: 2019-05-13
Playlist: WordCamp Kuala Lumpur 2018
Description: 
	This talk describes an ongoing developmental project for building a bespoke theme for a faculty. It includes the description of project requirements, the planning stages, and the actual development. It highlights the uniqueness of building a theme for an academic setup, and shares some of the difficulties in managing projects amidst other workloads.
