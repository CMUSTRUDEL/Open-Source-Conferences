Title: José Ángel Vidania: Estilos de bloque para el nuevo editor de WordPress
Publication date: 2019-10-28
Playlist: WordCamp Pontevedra 2019
Description: 
	En esta charla explicaré cómo crear estilos para un bloque del nuevo editor de WordPress. Y es que desde la llegada de los bloques en la versión 5.0, podemos crear diferentes estilos visuales para un mismo bloque y que estos estilos se apliquen de forma inmediata a nuestro contenido.
