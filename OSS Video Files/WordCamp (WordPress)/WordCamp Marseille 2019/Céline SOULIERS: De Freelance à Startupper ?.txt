Title: Céline SOULIERS: De Freelance à Startupper ?
Publication date: 2020-02-04
Playlist: WordCamp Marseille 2019
Description: 
	Est-ce qu’il est possible de se lancer dans la création d’une entreprise avec une simple idée, une API ou quelques lignes de code en poche ? Quelles sont les solutions pour m’aider à me lancer ? Comment savoir si mon idée peut aboutir à une startup ? Est-ce que j’ai envie de me lancer dans l’aventure ? À partir d’exemples issus de l’Incubateur Belle de Mai, Céline tentera de vous éclairer sur ces sujets.
