Title: OSMC 2015 | Current State of Icinga - Icinga Team
Publication date: 2015-11-27
Playlist: OSMC 2015 | Open Source Monitoring Conference
Description: 
	Zur Konferenz wird das Icinga Team die Version 2.4 des Icinga 2 Core fertigstellen. Neben einer Vielzahl an Feature-Erweiterungen, liegt der Schwerpunkt dieser Version in der Bereitstellung einer umfangreichen API. Diese soll Anwendern und Addon-Entwicklern zum einen mehr Flexibilität und Integrationsmöglichkeiten schaffen. Durch Verwendung von Standards und Client Libraries wird zum anderen der Zugriff stark vereinfacht und erlaubt die Anbindung auch ohne tiefgehende Entwicklungskenntnisse. Darüber hinaus steht mit Icinga Web 2 auch das neue Webinterface in einer finalen Version zur Verfügung und erlaubt unter Verwendung von Modulen eine simple Erweiterung des Basis-Featuresets. Eine Vorstellung der aktuellen Features und Entwicklungen gehört ebenso zum Vortrag wie eine detaillierte Demo.
