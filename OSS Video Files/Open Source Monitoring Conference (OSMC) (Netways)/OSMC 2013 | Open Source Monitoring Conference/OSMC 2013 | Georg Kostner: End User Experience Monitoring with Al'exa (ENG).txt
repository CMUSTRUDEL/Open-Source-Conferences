Title: OSMC 2013 | Georg Kostner: End User Experience Monitoring with Al'exa (ENG)
Publication date: 2015-01-07
Playlist: OSMC 2013 | Open Source Monitoring Conference
Description: 
	Al'exa ist ein Python Modul, das Kapazitäten zur Verfügung stellt, um grafische User Interfaces mit Hilfe von Computer Vision zu automatisieren und zu testen, damit die Verfügbarkeit und die Leistung der Anwenderapplikationen, die zum Beispiel  über Citrix und Windows Terminal Server deployed werden, überprüft werden können.

Al'exa wurde auf Basis von OpenCV Vision Library zur Erkennung von Objekten auf Template-basierten Mustern, zur Identifizierung von UI-Elementen (Labels, Input-Felder, Buttons, Links, Trees,...) um mit ihnen wie normale User interagieren zu können, kreiert. Eine auf Ninja Python basierte IDE ermöglicht es solche Test Cases zu schaffen. Die Durchführung solcher Test Cases mittels Al'exa ist bereits NSClient++-kompatibel, damit Nagios, Icinga, Shinken sie organisieren können und die Ergebnis-  (OK, WARNING, CRITICAL) und die Performance Daten zu erhalten.
