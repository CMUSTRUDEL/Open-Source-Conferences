Title: OSMC 2013 | Andreas Ericsson: The future of Nagios (ENG)
Publication date: 2015-01-07
Playlist: OSMC 2013 | Open Source Monitoring Conference
Description: 
	Following up on last year's talk, the presentation will go in-depth about new features in Nagios 4.1 and 5.0. Covering features available in Nagios 4.1, the talk will also bring some light into planned features on the current roadmap.

In detail the talk will cover components like helper daemons, load distribution, root cause analysis, object extensions, service sets, changes in macro handling, runtime addition/deletion of objects and a few other things.
