Title: CSS Grids im Detail | Sven Wolfermann | IPC Spring 2018
Publication date: 2019-02-11
Playlist: International PHP Conference Spring 2018
Description: 
	CSS Grids ist die einfache native Art, Grid Layouts mit CSS zu erstellen. Aktuelle CSS-Grid-Systeme sind mit den Mitteln geschrieben, die CSS bisher zur Verfügung hatte, meist auf Float-Basis, mit all den Nachteilen. CSS Grid Layouts sind völlig neu und erlauben nun individuelle Raster nicht nur für Seitenlayouts, sondern auch auf Komponentenebene. Sven Wolfermann (maddesigns) zeigt im Vortrag nicht nur, welche grundsätzlichen Eigenschaften CSS Grids bieten, sondern geht auch auf Fallback-Strategien ein.
