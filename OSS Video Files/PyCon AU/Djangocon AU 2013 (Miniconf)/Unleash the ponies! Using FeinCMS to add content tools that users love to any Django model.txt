Title: Unleash the ponies! Using FeinCMS to add content tools that users love to any Django model
Publication date: 2013-07-12
Playlist: Djangocon AU 2013 (Miniconf)
Description: 
	Greg Turner
http://2013.pycon-au.org/schedule/30026/view_talk
Site editors often struggle with editing rich content and managing variations in layout in Django admin. By rich content we mean text, images, video, tables, and so on. Trying to squeeze all this content into a single rich-text editor is like trying to jam a magical pony into a dog kennel. A better approach is to use a rich content framework like FeinCMS. FeinCMS is a sensible, flexible framework which allows rich content of any shape
