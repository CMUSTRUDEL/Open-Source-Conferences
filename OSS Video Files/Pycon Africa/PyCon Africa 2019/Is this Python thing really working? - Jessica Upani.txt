Title: Is this Python thing really working? - Jessica Upani
Publication date: 2019-11-23
Playlist: PyCon Africa 2019
Description: 
	Efforts of community engagement and coding events are on a rise in Namibia. We have been doing this since 2015. Now with the challenge of barely any Python jobs existing in the country, are all these efforts really working? Are they worth it? How do we keep going, how do we keep the community motivated to learn more and more Python?
