Title: The MDN Browser Compat Data Project
Publication date: 2018-02-20
Playlist: Mozilla at FOSDEM 2018
Description: 
	Presented by Florian Scholz 

First presented at FOSDEM, Feb 3, 2018. 
(https://fosdem.org/2018/schedule/event/mozilla_mdn_browser_compat_data_project/)

This talk describes how the MDN team is creating a rich data set for browser compatibility information at:
 https://github.com/mdn/browser-compat-data
