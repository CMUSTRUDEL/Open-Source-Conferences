Title: Accessible.JS - Jemima Abu - CityJS Conf 2020
Publication date: 2020-10-03
Playlist: CityJS Conf 2020
Description: 
	Depending on how it’s used, JavaScript can either make or break the accessibility of a site. This talk aims to cover all the ways to ensure accessibility of a website that’s dependent on JavaScript. Specific points to be addressed include - fallback methods for browsers - avoiding obtrusive JavaScript - proper display of callback text such as errors and confirmation messages - handling user interactions - ensuring easy navigation
