OBS provides a wide range of feature that help packagers to ship their software to their users. This talk is showing some of the key features of OBS and how they help packagers to make their life easier. For example, did you know that you can setup OBS to fetch package sources directly from GitHub and build them?

OBS is a Rails-based web application, with a perl backend, that allows users to build and distribute packages for a wide range of distributions, like SUSE, Fedora, Debian, Ubuntu and ARCH Linux.
