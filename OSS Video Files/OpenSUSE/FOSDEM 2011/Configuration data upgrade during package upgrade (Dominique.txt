Title: Configuration data upgrade during package upgrade (Dominique
Publication date: 2011-02-09
Playlist: FOSDEM 2011
Description: 
	For a casual user, editing configuration files in /etc in an intimidating task. Manually managing configuration upgrade is even worse as it requires good knowledge to merge current configuration data with new data coming from new packages.Config::Model was designed to make user's life easier by providing a configuration GUI and handle configuration upgrades transparently.
