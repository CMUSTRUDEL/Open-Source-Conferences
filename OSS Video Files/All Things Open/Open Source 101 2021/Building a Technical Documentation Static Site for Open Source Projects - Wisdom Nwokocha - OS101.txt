Title: Building a Technical Documentation Static Site for Open Source Projects - Wisdom Nwokocha - OS101
Publication date: 2021-04-10
Playlist: Open Source 101 2021
Description: 
	Wisdom Nwokocha, Mobile/Documentation Engineer at Savics, delivered the talk "Building a Technical Documentation Static Site for Open Source Projects" at the Open Source 101 conference on Tuesday, March 30, 2021.

Abstract:
Documentation is the product of technical writing that refers to different documents with product-related data and information; it has the details about a technical product that is either under development or already in use.

During this presentation, Wisdom will be talking about the tools that are used to build a static open source documentation site and host them on github pages/ Netlify.

Expect to come out of the session with:
an overview on tools such as 11ty, Docsify, docusaurus, MkDocs.
build and host a static website using 11ty framework on Netlify
