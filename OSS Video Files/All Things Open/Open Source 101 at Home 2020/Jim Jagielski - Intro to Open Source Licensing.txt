Description: 
	Intro to Open Source Licensing

Presented at: Open Source 101 at Home 2020
Presented by: Jim Jagielski, Uber

Abstract: Open Source licensing can be a tricky and confusing topic. In this session you will learn about the goals and intents of open source licensing, the main differences between various open source license types, and which license is best for your open source project. Whether consuming or contributing to Open Source, you need to know and understand basic open source licensing requirements.
