Title: Apollo Day SF 2018 - Opening Keynote (Matt DeBergalis)
Publication date: 2018-06-05
Playlist: Apollo Day SF 2018
Description: 
	From Apollo Day SF (May 31, 2018)

Apollo's cofounder and VP Engineering Matt DeBergalis presents a vision of Apollo as full-stack tooling that transforms how product engineering teams can build applications. See previews of new features like GraphQL schema validation and an integration with GitHub, the upcoming Apollo Server 2 including a replacement library for dataloader that provides partial query caching, and a preview of Apollo Server running inside Cloudflare's edge execution platform.
