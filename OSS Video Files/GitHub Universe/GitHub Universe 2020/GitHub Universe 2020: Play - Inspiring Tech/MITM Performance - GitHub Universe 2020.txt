Description: 
	Presented by NSDOS

MITM is a performance of live coding using ORCA, an esoteric programming language. Linked to his gears (synthesizers, drum machines, modulars, mainly product from Teenage Engineering) NSDOS will generate an algorithm with Orca to create an electronic music and visual piece of art. He can also trigger his algorithm with a motion sensor tied on his hand.
