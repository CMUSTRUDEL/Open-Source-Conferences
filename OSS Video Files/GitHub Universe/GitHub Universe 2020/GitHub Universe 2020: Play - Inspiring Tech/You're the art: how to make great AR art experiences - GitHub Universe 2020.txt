Title: You're the art: how to make great AR art experiences - GitHub Universe 2020
Publication date: 2020-12-12
Playlist: GitHub Universe 2020: Play - Inspiring Tech
Description: 
	Presented by Ash Herr, Artist, EMPOWA

In this session, multimedia artist Ash Herr explores how the integration of AR in our social platforms has created a new art form. She then breaks down how you can create an AR experience that is is engaging & comfortable. Topics include:

screen consideration
environment situations
illusion factor
audio enhancements
motion control
object interactions
