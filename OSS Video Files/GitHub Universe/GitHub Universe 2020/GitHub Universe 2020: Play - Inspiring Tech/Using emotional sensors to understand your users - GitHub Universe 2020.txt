Title: Using emotional sensors to understand your users - GitHub Universe 2020
Publication date: 2020-12-12
Playlist: GitHub Universe 2020: Play - Inspiring Tech
Description: 
	Presented by Elliott Bruce Hedman, Head Design Researcher, mPath

In this talk, Dr. Elliott Hedman will use psychophysiological sensors to measure a customer's subconscious and emotional reactions to products. He'll show you how product design improves when you know the exact moments your customers get stressed, excited, and confused.
