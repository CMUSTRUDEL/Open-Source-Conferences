Title: OSDC 2013 | Jan-Piet Mens "Ansible: configuration management doesn't have to be complicated"
Publication date: 2015-01-13
Playlist: OSDC 2013  | Open Source Data Center Conference
Description: 
	Jan-Piet will show how to quickly get started using Ansible for ad-hoc tasks, discuss some of its modules and introduce you to Ansible's playbooks and variables. He shows you how to run Ansible as a normal user (non-root), how to configure inventory data, and give you sundry tips on using Ansible effectively. If you prefer a pull-based setup, he will show you how to implement that as well.
