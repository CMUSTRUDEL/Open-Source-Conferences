Title: OSDC 2012 | Constantino Vázques: OpenNebula Open Source Toolkit for DataCenter Virtualization (EN)
Publication date: 2015-01-14
Playlist: OSDC 2012 | Open Source Data Center Conference
Description: 
	Open Nebula ist Standard in der Open Source Industrie, wenn es um die Visualisierung von Rechenzentren geht. Der Vortrag wird die innovativen Features aufzeigen, um virtualisierte Data Center umfassend und vollständig verwalten zu können und so IaaS-Clouds on-premise in bestehenden Infrastrukturen zu aktivieren. Die Interoperabilität von OpenNebula treibt die Cloud weiter voran, indem sie bestehende IT-Assets einsetzt, die Investitionen schützt und den Vendor Lock-in vermeidet.
