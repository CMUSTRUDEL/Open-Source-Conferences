Title: How Volkswagen and Rakuten Use Cloud Foundry For Multi-Cloud
Publication date: 2016-09-22

	Most organizations today are managing multiple clouds, but it can be difficult. Volkswagen, Rakuten, and PaaS providers explain how the Cloud Foundry platform can be used strategically for multi-cloud - allowing organizations to choose the right cloud for the right workload, create redundancy, avoid vendor lock-in, and focus on applications instead of infrastructure. https://www.cloudfoundry.org/
