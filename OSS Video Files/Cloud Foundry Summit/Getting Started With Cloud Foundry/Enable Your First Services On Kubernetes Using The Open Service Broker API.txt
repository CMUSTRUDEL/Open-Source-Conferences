Title: Enable Your First Services On Kubernetes Using The Open Service Broker API
Publication date: 2020-12-15
Playlist: Getting Started With Cloud Foundry
Description: 
	The use of service brokers completely eliminates the operational burden of provisioning a mixed spectrum of services required by applications. The complexity of maintaining these services across multiple infrastructure providers is reduced to zero.

The Open Service Broker API is a simple way to allow apps to get access to services that they need.

Learn more at openservicebrokerapi.org
